# EV-Charging-Demand-Forecasting-Customer-Segmentation-Machine-Learning-
This capstone project focuses on analyzing electric vehicle (EV) charging station usage in Warsaw to understand charging demand patterns, station performance, and user behavior. With the rapid adoption of electric vehicles, efficient charging infrastructure has become critical for sustainable urban transportation. The goal of this project was to analyze EV charging data and generate insights that can help optimize charging station utilization and improve infrastructure planning.

The project integrates multiple datasets related to EV charging sessions, station locations, and usage metrics. Using Python (Pandas and NumPy), the raw datasets were cleaned, merged, and preprocessed. This included handling missing values, removing duplicates, and performing feature engineering to prepare the data for analysis.

Advanced SQL queries were used to explore charging session patterns, identify peak usage hours, analyze station-level performance, and measure charging demand trends across different locations. Techniques such as joins, aggregations, subqueries, CTEs, and window functions were applied to extract meaningful insights from the data.

To effectively communicate these insights, an interactive Power BI dashboard was developed to visualize charging trends, station utilization rates, peak charging periods, and geographic distribution of charging stations. These visualizations help stakeholders easily understand usage patterns and identify opportunities to optimize EV charging infrastructure.

Overall, this project demonstrates how data analytics can support smarter urban mobility planning by helping city planners and energy providers make data-driven decisions to expand EV charging networks, reduce congestion at charging stations, and promote sustainable transportation.
